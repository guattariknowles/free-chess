# Free Chess 1.3.0

构建日期：2026-06-12

## 新增功能

- 首次本地对战前必须创建至少两个本地玩家档案。
- 支持创建、重命名和删除档案，普通对局记录双方档案。
- 新增自由局面编辑器，支持摆子、移动、删除、轮次、王车易位权利、
  吃过路兵格和回合计数。
- 支持 FEN 导入、复制、合法性校验和自定义局面本地保存。
- 支持 BO2、BO3、BO4、BO5、BO6 本地系列赛。
- 系列赛首局随机颜色，之后逐局换色；偶数局标记为公平赛制。
- 系列赛按胜 1 分、和 0.5 分、负 0 分计分。
- 主赛同分后自动进入两局 10+5 快棋、两局 3+2 超快棋和单局
  3+2 决胜。
- 系列赛整组保存在棋谱库，可查看总比分、全部单局和单局回放。
- 系列赛保存当前棋谱与暂停后的棋钟，重新打开 App 后可继续。

## 技术变更

- 新增本地档案、自定义局面和系列赛三个独立数据模块。
- 扩展棋谱记录，增加档案、初始 FEN 和系列赛关联信息，同时兼容
  旧版棋谱。
- 棋谱库将系列赛作为整体展示，系列赛单局不重复出现在独立棋局中。
- 自动化测试扩展到档案名称、FEN 校验、颜色轮换、积分和全部加赛阶段。
