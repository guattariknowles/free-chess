# 开发路线图

## 已完成

### 阶段 1：本地棋盘

- 点击走棋与合法走法高亮
- 完整标准规则判断
- 将军、将死与和棋
- 悔棋、重开与棋盘翻转
- 对局结束弹窗

### 阶段 2：棋钟

- 无棋钟、固定时间和自定义时间
- 每步加秒
- 暂停与继续
- 超时判负
- 双方独立操作区
- 认输确认

### 阶段 3：PGN 与棋谱

- 自动生成和复制 PGN
- PGN 导入与错误提示
- 本地保存对局
- 本地棋谱库
- 历史对局回放

## 当前阶段

### 阶段 4：本地档案与局面

- 本地用户档案
- 自定义起始局面

## 后续阶段

1. 基础规则、开局、中局和残局教学
2. 简单本地 AI
3. 小规模挑战与训练数据
4. Android 本地 Stockfish
5. 离线棋局分析
6. 棋盘与棋子皮肤

## 固定边界

- Android first
- Offline first
- 不做账号、在线匹配或云同步
- 不做广告和用户追踪
- 不依赖付费 API 或云端大模型
- 不复制第三方平台的题库、课程、素材或文案
