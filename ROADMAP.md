# 开发路线图

## 已完成

### 阶段 1：本地棋盘

- 点击走棋与合法走法高亮
- 完整标准规则判断
- 将军、将死与和棋
- 悔棋、重开与棋盘翻转
- 对局结束弹窗

### 阶段 2：棋钟

- 无棋钟、固定时间和自定义时间
- 每步加秒
- 暂停与继续
- 超时判负
- 双方独立操作区
- 认输确认

### 阶段 3：PGN 与棋谱

- 自动生成和复制 PGN
- PGN 导入与错误提示
- 本地保存对局
- 本地棋谱库
- 历史对局回放

### 阶段 4：本地档案与局面

- 创建、重命名和删除本地用户档案
- 普通本地对局必须选择两个不同档案
- 自定义起始局面、FEN 导入导出和本地保存
- 两个不同本地档案之间可创建 BO2 至 BO6 系列赛
- 系列赛首局颜色抽签，此后逐局交换黑白方
- 按 FIDE 标准积分记录总比分：胜 1 分、和棋 0.5 分、负 0 分
- 将系列赛及其全部单局作为一个整体保存在本地棋谱库
- 固定局数结束后总分高者获胜；同分进入预先选定的快棋、超快棋加赛
- 提示 BO3、BO5 无法让双方白棋局数完全相等，并提供偶数局公平赛制
- 系列赛保存当前棋谱与暂停棋钟，重开 App 后可继续
- 主页面改为左侧工具侧边栏，棋盘顶部只保留品牌与对局摘要
- 增加“对局 / 棋谱 / 玩家 / 我的”底部导航
- 自定义局面统一执行合法性检查，并显示错误列表与问题格标记
- 将死、逼和等已结束局面不能作为普通对局起点

## 固定边界

- Android first
- Offline first
- 不做账号、在线匹配或云同步
- 不做广告和用户追踪
- 不依赖付费 API 或云端大模型
- 不复制第三方平台的题库、课程、素材或文案
