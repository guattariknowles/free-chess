# 开发路线图

## 已完成

### 阶段 1：本地棋盘

- 点击走棋与合法走法高亮
- 完整标准规则判断
- 将军、将死与和棋
- 悔棋、重开与棋盘翻转
- 对局结束弹窗

### 阶段 2：棋钟

- 无棋钟、固定时间和自定义时间
- 每步加秒
- 暂停与继续
- 超时判负
- 双方独立操作区
- 认输确认

### 阶段 3：PGN 与棋谱

- 自动生成和复制 PGN
- PGN 导入与错误提示
- 本地保存对局
- 本地棋谱库
- 历史对局回放

### 阶段 4：本地档案与局面

- 创建、重命名和删除本地用户档案
- 普通本地对局必须选择两个不同档案
- 自定义起始局面、FEN 导入导出和本地保存
- 两个不同本地档案之间可创建 BO2 至 BO6 系列赛
- 系列赛首局颜色抽签，此后逐局交换黑白方
- 按 FIDE 标准积分记录总比分：胜 1 分、和棋 0.5 分、负 0 分
- 将系列赛及其全部单局作为一个整体保存在本地棋谱库
- 固定局数结束后总分高者获胜；同分进入预先选定的快棋、超快棋加赛
- 提示 BO3、BO5 无法让双方白棋局数完全相等，并提供偶数局公平赛制
- 系列赛保存当前棋谱与暂停棋钟，重开 App 后可继续
- 主页面改为左侧工具侧边栏，棋盘顶部只保留品牌与对局摘要
- 增加“对局 / 棋谱 / 玩家 / 我的”底部导航
- 自定义局面统一执行合法性检查，并显示错误列表与问题格标记
- 将死、逼和等已结束局面不能作为普通对局起点

### 阶段 5：基础教学

- 面对面双人模式下，反方棋子按反方视角显示为正向
- 增加 31 节原创离线课程，覆盖规则、棋钟、PGN、开局、中局和残局
- 教学详情使用本地文本和小棋盘展示 FEN 局面
- 进阶课程包含推荐走法、原因和常见错误解释
- 常见开局覆盖意大利、西班牙、后翼弃兵、伦敦、西西里和法兰西
- 中局覆盖子力安全、王安全、中心、开放线和双重攻击
- 残局覆盖王兵、基础车兵、后王杀单王和车王杀单王

### 阶段 6：简单本地 AI

- 新手 AI 随机选择合法走法
- 初级 AI 有吃子机会时优先吃子
- 中级 AI 使用一步子力评分，并优先完成一步将死
- 真人可选择执白、执黑或随机执色
- 人机对局兼容棋钟、投降、悔棋、棋谱保存和回放
- AI 只从合法走法中选择，不依赖网络、云端模型或 Stockfish

### 阶段 7：互动教学与分类栏 UI 修订

- 修复教学分类栏随课程数量变化而被拉高的问题
- 建立独立互动课程运行框架
- 保留全部 31 节原创课程和静态说明
- 26 节带推荐走法的课程支持实际走棋、反馈、提示、撤回和重来
- 5 节纯知识课程继续使用静态讲解
- 意大利开局、中心控制和王兵残局使用定制互动流程
- 当前对手使用离线本地 AI，并支持课程允许走法约束
- 课程运行层预留后续 Stockfish 替换位置

### 阶段 8：AI 对弈挑战、训练数据与 PGN 导入

- 全部 41 节内置课程都提供真人对离线 AI 的走棋教学
- 内置课程统一从标准初始局面开始，专题课在基础开局后进入目标局面
- 新增 7 节原创挑战，覆盖开局、中局战术和残局判断
- 新增歌剧院、常青树、不朽对局 3 节经典名局课程
- 经典棋谱只使用公开历史走法事实，全部教学解释为项目原创
- 每节内置课程包含可验证的目标、标签、来源和训练形式
- 增加本地 PGN 训练导入界面，支持开局、经典名局和残局
- 开局导入按棋谱训练前 6 个完整回合，之后由 AI 自主对弈
- 经典名局导入严格按照 PGN 复现
- 残局导入使用棋谱末端可继续局面，直接由 AI 自主回应
- 支持训练白方或黑方；训练黑方时 AI 先走白方第一步
- 导入内容只保存在本机，可在课程详情中删除
- 当前自主回应使用阶段 6 的离线 AI，后续由 Stockfish 替换

## 当前阶段

### 阶段 9：Android 本地 Stockfish

- 状态：`1.9.0` GitHub 测试版已发布，等待 `arm64` 实体手机和 EAS
  preview 验收
- 用本地 Stockfish 替换普通人机和教学自由对弈的走法提供层
- 保持完全离线，不调用云端模型或付费 API
- 先建立统一异步 `ChessEngine` 接口，界面不直接依赖 Stockfish
- 支持取消、超时、过期结果丢弃、合法性复核和简易 AI 降级
- 固定棋谱与固定教学脚本保持原逻辑，课程受限回应不得偏离教学目标
- Android 原生模块必须支持 `arm64-v8a` 真机和 `x86_64` 模拟器
- x86_64 模拟器、100 局面、取消、棋钟、教学和飞行模式已通过
- 完整执行方案与进度见
  [阶段 9 Stockfish 执行计划](docs/STAGE_9_STOCKFISH_PLAN.md)

### 阶段 10：离线棋局分析

- 状态：`1.12.0` 本地测试 APK 已更新复盘报告 UI，等待实体手机验收
- 棋谱库继续支持本地 PGN 导入，导入后可直接进入复盘页
- 复盘页支持分析当前步、全局复盘、停止分析和查看复盘报告
- 复用统一 `ChessEngine` 接口，不让界面直接调用 Android 原生模块
- 主界面隐藏 Stockfish、Elo、每步耗时、AI 推荐和 AI 重合度等工程化文案
- 复盘过程改为进度条和百分比
- 报告页显示白方/黑方五档走法质量统计：精彩、最佳、稳健、疑问、错误
- 报告页显示推荐吻合率、胜率变化折线图和关键波动入口
- Stockfish 不可用或超时时不把简易 AI 结果伪装成深度分析
- 当前分类和胜率是基于本地引擎评分变化的简化估算，不声称复刻第三方平台算法

### 阶段 11：全软件主题与棋盘/棋子皮肤

- 状态：`1.12.0` 本地测试 APK 已更新图标和首套完整棋子 SVG，等待 Android 设备外观切换验收
- 新增全软件主题系统，首版内置 5 套 UI 主题：
  Minimal Tournament、Impressionist Study、Constructivist Boardroom、
  Steampunk Atelier 和 Cyber Anime
- 新增 12 套棋盘皮肤，覆盖极简赛事、木质经典、印象派、构成主义、
  至上主义、蒸汽朋克、赛博霓虹、文艺复兴、浮世绘、水墨、新艺术和
  原创二次元清爽风格
- 新增 5 套 Unicode 棋子皮肤，并预留后续图片棋子接口
- App 图标已替换为用户指定的骑士棋子图标
- 新增 `teaching-large` 首套 12 件教学棋子 SVG 资源
- 主对局页、棋盘、棋钟、棋钟设置和外观设置弹窗接入主题
- 外观选择使用 `AsyncStorage` 保存在本机，读取失败时回退到默认主题
- 不引入账号、联网、广告、云端模型或第三方素材

## 固定边界

- Android first
- Offline first
- 不做账号、在线匹配或云同步
- 不做广告和用户追踪
- 不依赖付费 API 或云端大模型
- 不复制第三方平台的题库、课程、素材或文案
